# 리더 포럼 출석 관리 시스템 설치 가이드

## 📋 시스템 개요

리더 포럼 출석 관리 시스템은 6개월 텀으로 선출되는 디렉터들을 위한 포럼 출석부 시스템입니다.

### 주요 기능
- **9개 포럼 클래스**: 내셔널 트레이너, PR 코디, 도어퍼슨, 비지터 호스트, 성장 코디, 멤버십 위원회, 교육 코디네이터, 멘토링 코디, 의장단
- **스탬프 카드 시스템**: 이미지와 같은 형태의 출석 체크
- **실시간 연동**: 구글 시트와 웹 페이지 실시간 동기화
- **어드민 관리**: 포럼 일정, 유저 등록, 책임자 등록, 링크 관리

## 🚀 설치 및 설정 단계

### 1단계: 구글 시트 생성

1. **새 구글 시트 생성**
   - Google Drive에서 새 구글 시트 생성
   - 시트 이름: "리더 포럼 출석 관리"

2. **시트 탭 생성**
   다음 5개의 탭을 생성하세요:
   - `Users` - 유저 정보
   - `Sessions` - 포럼 세션 정보  
   - `Attendance` - 출석 기록
   - `Managers` - 포럼 책임자 정보
   - `Links` - 포럼 링크 정보

3. **각 시트의 헤더 설정**

   **Users 시트:**
   ```
   id | region | chapter | email | pin | className | name | createdAt | stamps
   ```

   **Sessions 시트:**
   ```
   id | title | date | time | type | location | onlineLink | className | managerId | createdAt
   ```

   **Attendance 시트:**
   ```
   userId | sessionId | attended | timestamp
   ```

   **Managers 시트:**
   ```
   id | forumClass | name | email | phone | region | chapter | createdAt
   ```

   **Links 시트:**
   ```
   id | forumClass | linkType | url | description | createdAt
   ```

### 2단계: 구글 앱스 스크립트 설정

1. **구글 앱스 스크립트 프로젝트 생성**
   - [Google Apps Script](https://script.google.com) 접속
   - "새 프로젝트" 클릭

2. **코드 복사**
   - `leader-forum-google-apps-script.js` 파일의 내용을 복사
   - 구글 앱스 스크립트 편집기에 붙여넣기

3. **시트 ID 설정**
   ```javascript
   const SHEET_ID = 'YOUR_GOOGLE_SHEET_ID'; // 실제 구글 시트 ID로 변경
   ```
   - 구글 시트 URL에서 ID 부분을 복사하여 위 코드의 `YOUR_GOOGLE_SHEET_ID` 부분을 교체

4. **시트 초기화 함수 실행**
   - 구글 앱스 스크립트 편집기에서 `initializeSheets` 함수 선택
   - "실행" 버튼 클릭하여 시트 구조 생성

### 3단계: 웹 앱 배포

1. **웹 앱 배포**
   - 구글 앱스 스크립트에서 "배포" > "새 배포" 클릭
   - 유형: "웹 앱" 선택
   - 실행 권한: "나" 선택
   - 액세스 권한: "모든 사용자" 선택
   - "배포" 클릭

2. **웹 앱 URL 복사**
   - 배포된 웹 앱의 URL을 복사
   - 예: `https://script.google.com/macros/s/AKfycbx.../exec`

### 4단계: 웹 페이지 설정

1. **HTML 파일 수정**
   - `forumstamp-html.html` 파일 열기
   - 다음 부분을 찾아서 수정:
   ```javascript
   const GOOGLE_APPS_SCRIPT_URL = 'https://script.google.com/macros/s/YOUR_DEPLOYMENT_ID/exec';
   ```
   - `YOUR_DEPLOYMENT_ID` 부분을 실제 배포된 웹 앱 URL로 교체

2. **웹 서버에 업로드**
   - 수정된 HTML 파일을 웹 서버에 업로드
   - 또는 GitHub Pages, Netlify 등 무료 호스팅 서비스 사용

## 📊 데이터 관리

### 유저 등록
어드민 페이지에서 다음 정보로 유저를 등록할 수 있습니다:
- 지역명 (예: 서울, 부산, 대구 등)
- 챕터명 (예: 강남, 해운대, 중구 등)
- 이메일 주소
- 4자리 PIN 번호
- 포럼 클래스 (9개 중 선택)

### 포럼 세션 관리
- 세션 제목, 날짜, 시간 설정
- 진행 방식 (오프라인/온라인/하이브리드)
- 장소 또는 온라인 링크 설정
- 포럼 클래스별 세션 관리

### 출석 체크
- QR 코드 스캔을 통한 자동 출석 체크
- 어드민에서 수동 출석 관리
- 실시간 출석률 확인
